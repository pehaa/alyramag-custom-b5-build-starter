## Variables

Ouvrer [SassMeister](https://www.sassmeister.com/)

** Quelle couleur prendra body ? **

Exemple 1 :

```scss
$color: orange;
$color: red;

body {
  background: $color;
}
```

Exemple 2 :

```scss
$color: orange;
$color: red !default;

body {
  background: $color;
}
```

Exemple 2 :

```scss

  background: $color;
```
